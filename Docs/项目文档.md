# 图书馆座位预约系统 — 项目文档

## 项目概述

图书馆座位预约系统是一个基于 ASP.NET Core MVC 的 Web 应用，让学生可以通过浏览器在线查看座位空闲情况、预约座位、取消预约；管理员可以在后台管理座位和查看预约记录。系统采用体验账号切换方式，无需注册即可使用。

## 功能模块

- **首页**：展示体验学生列表，下拉框切换账号，Session 持久化
- **座位列表**：12 座位网格展示，支持按区域/楼层筛选，动态显示状态（空闲/已预约/使用中/维护中）
- **座位详情**：展示座位基本信息 + 当日预约时段列表
- **预约提交**：填写时段提交预约，含时间规则校验（30 分钟~4 小时）、冲突检测、未登录守卫、座位状态守卫
- **我的预约**：当前用户全部预约记录，支持取消（仅 Pending 且未开始的可取消）
- **管理员登录**：admin / admin123 登录，Session 持久化
- **预约管理**：全部预约记录表格，支持按日期/状态/用户筛选，管理员可取消预约
- **座位管理**：查看全部座位、添加座位（含唯一性校验）、切换状态 Available ↔ Maintenance
- **统计页**：实时计算座位总数、今日预约数、使用率、各区域座位分布
- **权限守卫**：管理端 [AdminAuth] 过滤器，未登录自动重定向登录页

## 路由设计

| 路由 | Controller | Action | 说明 |
|------|-----------|--------|------|
| `/` | Home | Index | 首页 |
| `/Home/SwitchUser` | Home | SwitchUser | 切换体验账号 |
| `/Seats` | Seats | Index | 座位列表 |
| `/Seats/Detail/{id}` | Seats | Detail | 座位详情 |
| `/Reservation/Create/{id}` | Reservation | Create | 预约提交 |
| `/Reservation/My` | Reservation | My | 我的预约 |
| `/Reservation/Cancel/{id}` | Reservation | Cancel | 取消预约 |
| `/Admin/Login` | Admin | Login | 管理员登录 |
| `/Admin/Logout` | Admin | Logout | 管理员登出 |
| `/Admin/Reservations` | Admin | Reservations | 预约管理 |
| `/Admin/CancelReservation/{id}` | Admin | CancelReservation | 管理员取消预约 |
| `/Admin/Seats` | Admin | Seats | 座位管理 |
| `/Admin/CreateSeat` | Admin | CreateSeat | 添加座位 |
| `/Admin/ToggleSeatStatus/{id}` | Admin | ToggleSeatStatus | 切换座位状态 |
| `/Admin/Stats` | Admin | Stats | 统计页 |

## 数据库设计

3 张表：StudentUsers、Seats、Reservations。采用 EF Core Code First 方式，首次运行自动建表并写入种子数据。

详见 `Database/README.md`。
