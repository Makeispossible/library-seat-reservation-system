## C. 必改项

### 必改 1：重新生成全部 9 个 HTML 页面

`prototype/static-v1/` 下需要创建以下 9 个文件，使用已存在的 `css/style.css` 作为共享样式：

| 文件 | 路由对应 | 说明 |
|------|----------|------|
| `index.html` | `/` | 用户首页，含账号切换 |
| `seats.html` | `/Seats` | 座位卡片列表 + 筛选 + 空状态 |
| `seat-detail.html` | `/Seats/Detail/{id}` | 单座位详情 + 时段列表 |
| `reservation-create.html` | `/Reservation/Create/{seatId}` | 表单提交 + 校验错误状态 |
| `my-reservations.html` | `/Reservation/My` | 预约记录列表 + 取消弹窗 + 空状态 |
| `admin/login.html` | `/Admin/Login` | 登录表单 + 错误状态 |
| `admin/reservations.html` | `/Admin/Reservations` | 全部预约表格 |
| `admin/seats.html` | `/Admin/Seats` | 座位管理表格 + 添加表单 |
| `admin/stats.html` | `/Admin/Stats` | 数据卡片 + 分布表 |

要求全部已在 `docs/05-页面卡与UI规范.md` 中定义。每个页面必须：
- 使用 Bootstrap 5 CDN（CSS + JS）
- 引用 `css/style.css`（相对路径）
- 用户端蓝色导航栏，管理端深色导航栏
- 页面间通过相对链接跳转
- 展示成功状态 + 至少一种空状态或错误状态
