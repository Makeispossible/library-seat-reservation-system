## 附录：Sprint 1-4 任务卡通用约束

所有业务代码任务卡必须包含以下验收维度：

| 维度 | 要求 |
|------|------|
| 编译 | `dotnet build` 无错误、无警告 |

| 异常 | 覆盖 `docs/09-关键链路详细设计.md` 中该模块的异常/边界表 |
| TempData | 操作结果通过 TempData 传递，在 Layout 中统一渲染 |
| 守卫 | 管理端 Action 标注 `[AdminAuth]`，用户端预约操作检查 Session |
