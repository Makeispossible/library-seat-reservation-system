## 3. @playwright/test 安装与 msedge 配置

### 安装命令

```bash
cd LibrarySeatReservation
npm init -y
npm install -D @playwright/test
```

安装结果：`@playwright/test@^1.52.0`，共 3 个包，用时 18s。

### playwright.config.ts 配置

```typescript
import { defineConfig } from '@playwright/test';

export default defineConfig({
  testDir: './e2e',
  timeout: 60000,
  retries: 0,
  use: {
    baseURL: 'http://localhost:5207',
    channel: 'msedge',          // 使用系统自带 Edge，不下载 Playwright 自带浏览器
    headless: true,
    screenshot: 'only-on-failure',
    trace: 'retain-on-failure',
  },
  webServer: {
    command: 'dotnet run --project src/LibrarySeatReservation.Web/LibrarySeatReservation.Web.csproj --urls http://localhost:5207',
    port: 5211,
    reuseExistingServer: true,  // 允许使用已启动的应用
    timeout: 60000,
  },
});
```

**关键配置说明：**
- `channel: 'msedge'`：使用系统自带的 Microsoft Edge，**不要求**下载 Playwright 自带的 Chromium
- `reuseExistingServer: true`：允许使用已启动的 `dotnet run` 实例，无需每个测试都启动
