## C. 必改项

### 必改 1：修正预约提交页的路由参数名

**位置**：第 6 节，第 4 行

**原文**：`/Reservation/Create/{seatId}`

**问题**：ASP.NET Core 默认路由 `{controller=Home}/{action=Index}/{id?}` 中，URL 第三段固定绑定到参数名 `id`，`{seatId}` 不会自动匹配。

**解决方案（二选一，建议方案 A）：**

方案 A（推荐，改动最小）：将 Action 参数名改为 `id`

```
路由不变：/Reservation/Create/5
Controller: Create(int id)
View 链接：<a asp-route-id="@seat.Id">
```

方案 B：使用 `[Route]` 属性显式声明

```csharp
[Route("Reservation/Create/{seatId}")]
public IActionResult Create(int seatId)
```
