## C. 必改项（达到最低完成线的前提）

### C1. 修复 CancelReservation 日期 Bug（F-01）

**AdminController.cs:154** 当前代码：
```csharp
if (DateTime.Now.TimeOfDay >= reservation.StartTime)
```

应改为：
```csharp
if (reservation.Date == DateOnly.FromDateTime(DateTime.Today) && DateTime.Now.TimeOfDay >= reservation.StartTime)
```

**ReservationController.cs:195** 相同修改。

理由：未来日期的预约（date > today）无论当前时间如何，管理员/学生都应可以取消。只有今天的预约需要通过时间判断是否已开始。
