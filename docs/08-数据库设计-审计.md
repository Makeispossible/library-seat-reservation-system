## D. 建议优化项

### 建议 1：索引 IX_Reservations_Date_StartTime 可考虑包含 Status

```sql
-- 当前（不含 Status）
CREATE INDEX IX_Reservations_Date_StartTime ON Reservations (Date DESC, StartTime DESC);

-- 建议（包含 Status，覆盖管理端按状态筛选场景）
CREATE INDEX IX_Reservations_Date_StartTime ON Reservations (Date DESC, StartTime DESC) INCLUDE (Status);
```

### 建议 2：确认"今日预约数"统计口径

当前口径：`COUNT(*) WHERE Date = @today`（含 Cancelled）

本意应该只统计有效预约（Pending + 动态计算的使用中/已完成），排除 Cancelled：

```sql
SELECT COUNT(*) FROM Reservations WHERE Date = @today AND Status = 'Pending'
```

### 建议 3（可选）：Seat 表补充 CreatedAt 字段

如果 PRD-Lite 的 `Seat.CreatedAt` 需要在代码中用到（如管理端查看座位添加时间），可以在 Seat 模型中追加：

```csharp
public DateTime CreatedAt { get; set; } = DateTime.Now;
```
